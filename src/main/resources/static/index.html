Library Management API

Welcome to the Library Management API! This web application is designed to help manage books and authors, allowing users to perform CRUD (Create, Read, Update, Delete) operations on a collection of books and their respective authors.

The API provides endpoints for adding new books and authors, updating existing records, and viewing or deleting specific entries.

If you'd like to test the API endpoints directly, you can use Postman, a tool that allows you to easily send requests and see responses from the API.

*Developed as part of a learning project using Java, Spring Boot, and PostgreSQL.
